# STM32C0 看门狗 (IWDG) 说明

> 版本：v1.0 | 创建：2026-03-23

---

## 与旧 MCU (F103) 的差异

| 项目 | 旧 MCU (F103) | 新 MCU (STM32C0) |
|------|---------------|------------------|
| Window 寄存器 | 无 | 有 IWDG_WINR |
| Window=0 | - | 禁用窗口模式，随时可喂狗 |
| 超时公式 | T = (Prescaler × Reload) / LSI | 同左，LSI 约 32 kHz |

---

## 当前配置 (v2.5.1)

| 参数 | 值 | 说明 |
|------|-----|------|
| Prescaler | 256 | 分频系数 |
| Reload | 4095 | 12 位最大值 |
| Window | 0 | 禁用窗口 |
| 超时 | ~32.7 s | 256×4095/32000 |

---

## 关键点

1. **启动时机**：`MX_IWDG_Init()` 必须放在主循环入口前、所有耗时初始化之后，否则 init 阶段易超时复位。
2. **Window=0**：与旧 MCU 无窗口时的行为一致，喂狗时机不受限。
3. **超时余量**：原 Prescaler=64、Reload=1000 理论上约 2 s，实测出现 ~130 ms 复位，可能与 LSI 偏差有关；改为 256/4095 预留充足余量。
