---

## 2. 移植意图

- 将旧 MCU（F103）的完整功能迁移到新 MCU（C011），功能保持一致
- 采用**积木化**方式：逐块移植、逐块验证
- 通讯（2.4G XL2400T）为最核心模块，**已移植成功**
