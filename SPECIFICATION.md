# 太阳能供电同步闪烁路灯 - 规格书

> 版本：v1.2.0 | 创建：2025-03-09 | 更新：2026-03-14

---

## 1. 项目概述

太阳能供电的同步闪烁路灯系统，白天由太阳能板为镍氢电池充电，夜间自动检测并开启同步闪烁照明。

---

## 2. MCU 选型

| 阶段 | 型号 | 封装 | 说明 |
|------|------|------|------|
| 开发期 | STM32F103ZET6 | LQFP144 | 现有开发板，便于快速开发 |
| 量产期 | STM32C011F4P6 | TSSOP-20 | 低成本，16KB Flash，支持 Stop/Standby 低功耗 |

**软件可移植**：开发用 F103、量产用 C011，需硬件抽象或条件编译，确保可移植。

---

## 3. 无线同步

- **模块**：深圳市芯岭技术 XL2400T（2.4G）
- **方式**：对等同步（无主节点，各模块硬件一致）
  - 每周期约 50ms 前进入短 RX 窗口；收到 sync 则对齐定时器且不发送；未收到且本机到点则发送 sync 并闪灯
  - 自然涌现发起者，每周期仅 1 次 TX，其余只 RX
- **规模**：几百个、直线/堆叠、间距几米、单跳几十米 → 邻节点重叠，sync 逐跳传播，5～10 周期内整网收敛

---

## 4. 电源系统

| 项目 | 规格 |
|------|------|
| 电池 | 1.2V 镍氢电池，800mAh |
| 太阳能板 | 2.4V 太阳能电池板 |
| 供电链路 | 1.2V NiMH → DC-DC 升压至 3.3V (MCU) / LED 升压驱动 |
| 充电 | 太阳能板直充 + MOSFET 控制，充满后关断 |

**过放/过充与滞回：**

| 项目 | 阈值 | 动作 |
|------|------|------|
| 过放 | < 1.0V | 进入低功耗模式，停止闪烁 |
| 过放恢复 | > 1.1V | 退出低功耗，恢复闪烁 |
| 过充 | ≥ 1.45V | 关断充电 MOSFET |
| 充电滞回 | < 1.35V | 重新开启充电（避免 1.45V 边界抖动） |

---

## 5. 日/夜检测

| 项目 | 规格 |
|------|------|
| 检测对象 | 太阳能板输出电压 (PA0) |
| 开启 | < 0.29V → 开启闪灯 |
| 关闭 | > 0.4V → 关闭闪灯 |
| 说明 | 滞回 0.11V，避免临界抖动 |

**采样方式**：太阳能板电压 0.29V～0.4V 直接接入 ADC，不分压、不放大（低成本）。

---

## 6. 闪灯逻辑

| 项目 | 规格 |
|------|------|
| 亮 | 100ms |
| 灭 | 800ms |
| 周期 | 900ms |

## 7. LED 驱动规格（v1.2.0 更新）

### 7.1 LED 配置
- **数量**：6 颗白色 LED 并联
- **型号**：TJ-S3014UG

### 7.2 驱动方式：电感升压驱动

| 参数 | 值 |
|------|-----|
| 升压电感 | L1 = 33μH |
| 开关管 | Q3 = CJ2302 (N-MOS) |
| PWM 频率 | **124kHz** |
| PWM 占空比 | **60%** |

### 7.3 控制引脚

| 引脚 | 名称 | 功能 | 控制逻辑 |
|------|------|------|----------|
| PA2 | LED_DRV | PWM 输出驱动升压 | 输出 PWM → LED 亮；LOW → LED 灭 |
| PA3 | BOOST_EN | 升压电路供电使能 | MCU 启动后 100ms 置 HIGH，始终保持 |

### 7.4 LED 控制时序

```
MCU 启动：
  初始化 → 时钟稳定 → 延时 100ms → PA3 (BOOST_EN) = HIGH

闪烁周期 (900ms)：
  [0-100ms]   PA2 输出 124kHz 60% PWM → LED 亮
  [100-900ms] PA2 = LOW (停止 PWM)    → LED 灭
  
PA3 始终保持 HIGH（不随 LED 亮灭变化）
```

## 8. 低功耗策略

| 项目 | 措施 |
|------|------|
| 电池电压检测 | PA1 (ADC) 检测，预留功能 |
| 充电 | MOSFET 控制，充满关闭 |
| MCU | 闪灯间隙 Stop/Sleep |
| XL2400T | 非收发时休眠（约 1.53μA） |
| LED 升压 | 灭灯时停止 PWM，升压电路不工作 |
| 目标 | 压低整机平均电流，应对连续多日阴天 |

## 9. IO 分配（STM32CubeMX 参考）

### 9.1 开发板 STM32F103ZET6

| 功能 | 名称 | 引脚 | 类型 | 说明 |
|------|------|------|------|------|
| 太阳能检测 | Solar_ADC | PA0 | ADC1_IN0 | 日/夜检测，0.29V/0.4V 滞回 |
| 电池检测 | Batt_ADC | PA1 | ADC1_IN1 | **预留**，过充/过放检测 |
| LED 驱动 | LED_DRV | PA2 | **TIM2_CH3 PWM** | 124kHz 60% PWM |
| 升压使能 | BOOST_EN | PA3 | GPIO OUT | 升压电路供电控制 |
| RF 片选 | RF_CSN | PA4 | GPIO OUT | XL2400T 软件 SPI |
| RF 时钟 | RF_SCK | PA5 | GPIO OUT | XL2400T 软件 SPI |
| RF 数据 | RF_DATA | PA7 | GPIO I/O | XL2400T 软件 SPI |
| 充电控制 | CHG_MOS | PA8 | GPIO OUT | 充电 MOSFET 控制 |
| 调试串口 | USART1_TX | PA9 | USART | 调试输出 |
| 调试串口 | USART1_RX | PA10 | USART | 调试输入 |
| 分压控制 | DIV_MOS | PA11 | GPIO OUT | 电池分压 MOSFET 控制 |
| 状态指示 | LED | PB5 | GPIO OUT | 板载 LED |

PA13/PA14 保留 SWD 调试。

### 9.2 PWM 配置参数（PA2 / TIM2_CH3）

| 参数 | 值 | 计算 |
|------|-----|------|
| 系统时钟 | 24MHz | HSE 8MHz × PLL |
| 目标频率 | 124kHz | - |
| Prescaler | 0 | 不分频 |
| Counter Period (ARR) | 192 | 24MHz / 124kHz - 1 |
| Pulse (CCR) | 115 | 192 × 60% ≈ 115 |
